# AI Customer Support Workflow (n8n)

## Overview
AI-powered customer support automation workflow built using n8n.

The system automatically processes incoming customer emails, classifies requests using AI, retrieves contextual knowledge through a RAG pipeline, and generates intelligent automated responses.

---

## Features
- Gmail email automation
- AI-powered email classification
- Automated customer response generation
- Retrieval-Augmented Generation (RAG)
- Pinecone vector database integration
- Ollama embeddings
- Context-aware AI replies
- Automated workflow routing

## Workflow Process
1. Incoming customer email is detected
2. AI classifier categorizes the message
3. Relevant information is retrieved using semantic search
4. Pinecone returns contextual knowledge
5. AI agent generates a response
6. Automated reply is sent to the customer

---

## Import Workflow
1. Download the workflow JSON
2. Import into n8n
3. Configure Gmail credentials
4. Connect Pinecone and Ollama
5. Add OpenRouter API key
6. Run the workflow
